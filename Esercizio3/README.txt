***Compilation***

---to compile and execute---
    1) go to .../UnionFindSet
    2) ant

---to delete the .class files and the .jar archive---
    1) go to .../UnionFindSet
    2) ant clean

---to compile the unit tests---
    1) go to .../UnionFindSet
    2) ant junit


***Execution***

---to execute the automatic tests---
    1) go to .../UnionFindSet
    2) java -jar build/UnionFindSet.jar
